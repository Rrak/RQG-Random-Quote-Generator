# RQG-Random-Quote-Generator

<p>Simple quote generator which generates new quote with on click event initiated by user, it uses QoD[Quotes on Design API] which acquires data in JSON format.</p>

<h3> Built Using
<ul>
<li>Jquery</li>
<li>Animate.css</li>
<li>Font-Awesome</li>
<li>Twitter API</li>
<li>QoD API</li>
</ul>
